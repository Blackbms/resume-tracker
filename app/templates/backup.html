CLI Commands

You can also manage backups from the command line:

# Export to JSON
flask backup export-json

# Export to SQL
flask backup export-sql

# Export both formats
flask backup export-all

# Import from JSON
flask backup import-json path/to/backup.json

# Import from SQL
flask backup import-sql path/to/backup.sql

# List existing backups
flask backup list

# Custom output path
flask backup export-json --output /custom/path/backup.json

Backups are stored in the backups/ directory by default.
