# Ideas & Future Features

A running list of possible improvements to the resume tracker. No commitment to build any of these — just a place to capture them.

---

## Job Hunting Workflow

- [ ] Email reminders for applications with no response after X days
- [ ] Follow-up date field with a visual "needs follow-up" indicator on the list
- [ ] Notes history / timeline per application (currently notes are overwritten on edit)
- [ ] Dashboard with application counts grouped by status
- [ ] Track where resume has been sent
- [ ] Job Portal Tracking
- [ ] Recruiter tracking
  - [ ] Name
  - [ ] Phone
  - [ ] Email
  - [ ] Company URL
  - [ ] Linked In URL
      

## Data & Reporting

- [ ] Export applications to CSV
- [ ] Response rate stats (% that reach interview, offer, etc.)
- [ ] Chart of applications submitted over time

## Usability

- [ ] Search / filter by company name or role keyword
- [ ] Filter by status on the index page
- [ ] Pagination for large application lists
- [ ] Mobile-friendly layout improvements

## Auth & Multi-User

- [ ] User login so the app can be hosted publicly
- [ ] Support for multiple users / separate job search campaigns

## Integrations

- [ ] Auto-populate company and role by pasting a job posting URL
- [ ] LinkedIn / Indeed link parsing
- [ ] Calendar integration for interview dates
